# server-app-udemy

Implementation of one of the exercises in The Complete Node.js Developer Course (2nd Edition) by Andrew Mead.
Shows concept of Web Server, using Express, Templating Engine.. etc.

## Tech Stack
- Node.js v8+
- Express
- Handlebars
- Heroku

## What is it
A very simple web app

## How to run
1. npm install
2. node server.js
